==Welcome to WDI-CONF!
===============================
To set up:

fork the repo to your own github repo.
clone from your own github to your local computer.
Then set up locally:

configure .gitignore
configure your database.yml and add to .gitignore
bundle install
Do whatever mac thing you need to do.
rake db:create
rake db:migrate
rails server (or rails s)
in browser go to localhost:3000 to see if it all worked.


=== About the site:
WDI Conf is a website created for a semi-fictional tech conference. The site was based on current design trends found on many conference websites, namely; centre-aligned text, subtle animations to guide you through the site and a simple colour palate.
